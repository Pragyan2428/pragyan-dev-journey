# Pragyan's Dev Journey

A personal developer productivity dashboard built with HTML, CSS, and JavaScript to track learning progress, projects, skills, and daily coding habits.

## Live Demo

https://Pragyan2428.github.io/pragyan-dev-journey/

## GitHub Repository

https://github.com/Pragyan2428/pragyan-dev-journey

---

## Overview

Pragyan's Dev Journey is a personal dashboard designed to help aspiring developers stay consistent, track progress, manage projects, and build productive coding habits.

The application runs entirely in the browser and uses Local Storage to save user data without requiring a backend.

---

### Developer Dashboard

- Daily progress tracking

### Learning Roadmap

- Automatic data saving
- Persistent progress tracking

## Running Locally

### Clone the repository

```bash
git clone https://github.com/Pragyan2428/pragyan-dev-journey.git
```

### Navigate to the project folder

```bash
cd pragyan-dev-journey
```

### Open the application

Open `index.html` in any modern web browser.

No installation or dependencies are required.
